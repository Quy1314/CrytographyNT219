
0. Download Openssl
https://github.com/openssl/openssl/releases/download/openssl-3.5.4/openssl-3.5.4.tar.gz
and extract the gz file to obtain the source folder: openssl-3.5.4
1. Compile Openssl using g++ with mingw64 (run on mingw64 terminal in openssl-3.5.4)
```
cd C:\Tool\openssl-3.5.4\openssl-openssl-3.5.4
```
- Check for sure that only one make program in your computer: 
```
where make
```
the return should be one path to make.exe (C:\msys64\usr\bin\make.exe)
- Config compiler:
```
./Configure CC="/mingw64/bin/gcc" CXX="/mingw64/bin/g++" --prefix="C:/Tool/openssl-3.5.4/openssl-openssl-3.5.4/gcc"
```
- Compile the library
```
make clean -j 16
make -j 16
make test -j 16
make install -j 16
```
notes:
--prefix: location for install (copy) after compiling
-j <number>: the number CPU threads for parallel processing
